Les règles du Hangman sont simples : un mot est choisi au hasard, et le joueur doit le deviner lettre par lettre. À chaque erreur, une partie du pendu est dessinée. Si le joueur découvre toutes les lettres avant que le dessin soit complet, il gagne. Sinon, il perd lorsque le pendu est entièrement dessiné. Les lettres déjà essayées ne coûtent pas de tentative supplémentaire.

Choisir son niveau

La sélection de difficulté permet d'adapter le jeu au niveau du joueur. En mode facile, les mots sont courts et simples, idéal pour les débutants. Le mode moyen propose des mots plus complexes pour un défi modéré, tandis que le mode difficile offre des mots longs et rares pour les joueurs expérimentés. Un mode personnalisé permet également d'importer ses propres mots pour une expérience sur mesure.

Normal ? Infinite ?

Normal : Le joueur doit deviner un seul mot avec un nombre limité de tentatives (par exemple, 10). La partie se termine après une victoire ou une défaite. Ce mode est idéal pour des sessions courtes

Infinite : Le joueur enchaîne des mots sans limite tant qu’il reste des tentatives. Les points (succès) s’accumulent après chaque mot deviné correctement, offrant une expérience prolongée et compétitive.
